# 情侣空间 App 原型设计

这是一个基于 HTML + Tailwind CSS 的情侣空间 App 原型设计，模拟了真实的 iOS/Android 应用界面。

## 项目结构

```
.
├── index.html          # 主入口文件，包含手机外壳和底部导航
├── pages/             # 页面文件夹
│   ├── home.html      # 首页
│   ├── messages.html  # 消息页面
│   ├── gallery.html   # 相册页面
│   ├── diary.html     # 日记页面
│   ├── profile.html   # 个人资料页面
│   └── settings.html  # 设置页面
└── README.md          # 项目说明文档
```

## 功能特点

### 1. 首页
- 恋爱纪念日倒计时
- 功能快捷入口（相册、日记、约会、礼物）
- 最近动态展示

### 2. 消息
- 实时消息列表
- 系统通知
- 消息搜索
- 新建消息

### 3. 相册
- 照片分类展示
- 照片网格浏览
- 照片上传
- 相册管理

### 4. 日记
- 心情统计
- 日记列表
- 写日记
- 心情标签

### 5. 个人资料
- 用户信息展示
- 情侣信息
- 纪念日管理
- 数据统计

### 6. 设置
- 账号设置
- 通知设置
- 隐私设置
- 数据备份
- 其他设置

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome 图标
- 响应式设计

## 使用说明

1. 克隆项目到本地
2. 使用现代浏览器打开 `index.html`
3. 即可查看完整的 App 原型界面

## 设计特点

1. 真实感
   - 模拟 iPhone 15 Pro 外观
   - 圆角设计
   - 状态栏模拟
   - 底部导航栏

2. 交互体验
   - 流畅的页面切换
   - 直观的操作反馈
   - 符合移动端操作习惯

3. 视觉设计
   - 现代简约风格
   - 统一的色彩系统
   - 精美的图标设计
   - 合理的留白和间距

## 开发说明

本项目使用 Tailwind CSS 进行样式开发，所有页面都遵循相同的设计规范，确保视觉统一性。每个页面都是独立的 HTML 文件，通过 iframe 在主页面中展示，方便开发和调试。

## 注意事项

1. 本项目仅作为原型设计，不包含实际的后端功能
2. 图片资源使用 Unsplash 的示例图片
3. 所有交互效果需要在实际开发中实现

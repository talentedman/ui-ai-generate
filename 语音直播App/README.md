# 声动 - 语音直播应用原型

这是一个高保真的语音直播应用原型，使用HTML、CSS和JavaScript构建。原型模拟了iPhone 15 Pro的尺寸和UI风格，提供了完整的用户体验流程。

## 项目结构

```
├── css/            # CSS样式文件
│   └── styles.css  # 主样式表
├── js/             # JavaScript文件
│   └── app.js      # 应用交互逻辑
├── img/            # 图片资源
│   └── ...         # 各种图片资源
└── pages/          # HTML页面
    ├── splash.html        # 启动页
    ├── login.html         # 登录/注册页
    ├── home.html          # 首页
    ├── discover.html      # 发现页
    ├── live_room.html     # 直播间
    ├── profile.html       # 个人中心
    ├── create_live.html   # 创建直播
    ├── messages.html      # 消息中心
    ├── wallet.html        # 钱包/充值
    └── settings.html      # 设置页面
```

## 使用说明

1. 打开 `index.html` 文件查看所有页面的预览
2. 点击各个页面的缩略图可以查看对应页面的详细内容
3. 各页面之间可以通过导航栏和按钮进行跳转

## 功能概述

- **用户认证**: 登录、注册、第三方账号登录
- **内容浏览**: 热门直播、推荐主播、分类浏览
- **互动功能**: 点赞、评论、送礼物
- **创建直播**: 设置直播标题、封面、分类等
- **个人中心**: 用户资料、作品管理、关注列表
- **虚拟货币**: 钱包充值、礼物购买、收益管理
- **系统设置**: 通知设置、隐私设置、账号安全

## 界面预览

原型包含以下关键界面:

1. **启动页**: 展示应用品牌和加载动画
2. **登录/注册**: 用户认证界面
3. **首页**: 推荐内容和热门直播
4. **发现页**: 分类和排行榜
5. **直播间**: 实时互动、评论和礼物发送
6. **个人中心**: 用户信息和统计数据
7. **直播创建**: 设置直播参数
8. **消息中心**: 系统通知和私信
9. **钱包**: 虚拟货币管理和充值
10. **设置页**: 应用配置和用户偏好

## 技术栈

- HTML5
- CSS3 (含Flexbox和Grid布局)
- JavaScript (ES6+)
- Tailwind CSS
- Font Awesome 图标

## 开发注意事项

- 此原型仅作为UI/UX设计展示，不包含后端功能
- 原型针对移动设备进行了优化，特别是iPhone 15 Pro
- 使用了响应式设计，但最佳体验为移动设备尺寸

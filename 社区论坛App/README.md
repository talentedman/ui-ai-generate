# 社区论坛App原型

这是一个使用HTML、Tailwind CSS和Font Awesome构建的社区论坛App高保真原型。

## 项目结构

```
社区论坛App/
├── css/
│   └── main.css          # 主要样式文件
├── components/
│   └── template.html     # 页面模板
├── images/               # 图片资源目录
├── pages/               # 页面文件
│   ├── home.html        # 首页
│   ├── discover.html    # 发现页
│   ├── post.html        # 发布页
│   ├── message.html     # 消息页
│   ├── profile.html     # 个人资料页
│   ├── settings.html    # 设置页
│   ├── login.html       # 登录页
│   └── register.html    # 注册页
└── index.html           # 主入口文件
```

## 功能特点

- 响应式设计，适配移动设备
- 现代化的UI设计
- 完整的用户交互流程
- 高保真原型展示

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome
- Google Fonts (Noto Sans SC)

## 运行方式

1. 克隆项目到本地
2. 使用现代浏览器打开 `index.html` 文件
3. 所有页面将在主页面中以iframe形式展示

## 页面说明

- 首页：展示帖子列表和推荐内容
- 发现：展示热门话题和推荐用户
- 发布：发布新帖子的界面
- 消息：展示通知和私信
- 个人资料：展示用户信息和内容
- 设置：用户设置和系统配置

## 注意事项

- 这是一个静态原型，不包含后端功能
- 图片资源需要自行添加
- 建议使用现代浏览器访问以获得最佳体验

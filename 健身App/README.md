# 健身应用原型设计

这个项目是一个基于HTML、Tailwind CSS和FontAwesome的高保真健身应用原型。

## 项目简介

本项目是一个健身锻炼App的高保真原型设计，旨在展示应用的界面和交互逻辑，为实际开发提供参考。该原型设计遵循现代移动应用UI/UX设计规范，模拟了真实的iOS/Android应用界面。

## 技术栈

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (通过CDN引入)
- [FontAwesome](https://fontawesome.com/) (用于图标)

## 页面结构

项目包含以下主要页面：

1. **首页 (home.html)** - 展示用户每日活动、目标进度和推荐训练
2. **锻炼页 (workout.html)** - 提供不同类型的锻炼计划和运动项目
3. **进度追踪 (progress.html)** - 展示用户的锻炼数据统计和成就
4. **社区页 (community.html)** - 用户交流分享和专业指导
5. **个人资料 (profile.html)** - 用户信息和应用设置

所有页面都保持一致的设计风格，包括顶部状态栏和底部导航栏，以增强真实感和用户体验。

## 如何使用

1. 克隆或下载此仓库到本地
2. 直接在浏览器中打开`index.html`文件
3. index.html页面使用iframe展示所有原型界面，方便整体浏览和比较

## 设计特点

- **真实感** - 模拟iPhone 15 Pro的尺寸和界面样式
- **高保真UI** - 使用现代UI元素和真实图片资源
- **响应式设计** - 适配不同屏幕尺寸
- **符合设计规范** - 遵循iOS/Android的设计语言和交互模式

## 目标用户

- 产品经理 - 用于产品演示和需求讨论
- UI/UX设计师 - 作为设计参考和灵感来源
- 开发者 - 作为前端开发的直接参考和模板

## 后续开发

这个原型可以作为实际开发的基础，通过以下步骤进一步完善：

1. 将静态HTML转换为前端框架组件(如React, Vue等)

原型使用说明

设计规范

  • 设备尺寸：iPhone 15 Pro (390×844px)
  • 主色系：深蓝色 (#0052CC)、浅蓝色 (#4C9AFF)
  • 强调色：橙色 (#FF8800)、绿色 (#00875A)、红色 (#FF5630)
  • 字体系列：System Font (SF Pro Display/Inter)

页面结构

  • 顶部状态栏：展示时间、网络状态、电池电量
  • 内容区域：各页面主要内容
  • 底部导航栏：首页、账户、转账、理财、我的

开发说明

  • 所有页面基于HTML + Tailwind CSS + FontAwesome开发
  • 目录结构清晰：css/、components/、pages/
  • 组件化设计：状态栏、导航栏等为独立组件
  • 所有原型均支持基本的交互操作

# 天气App高保真原型

这是一个基于HTML、Tailwind CSS和FontAwesome的天气App高保真原型项目。该原型完整模拟了移动端天气应用的用户界面和交互设计。

## 项目概述

本项目使用纯前端技术构建，包含了多个页面和完整的交互设计。原型完全模拟了iOS设计风格，提供了接近真实App的用户体验。

## 主要功能

- **首页**：展示当前城市的天气概况、今日天气详情和未来天气预报
- **详情页**：展示更详细的气象数据，包括每小时预报、气象详情和日出日落时间
- **城市管理**：管理已添加的城市，并可以添加新城市
- **搜索页**：搜索并添加新城市
- **设置页**：调整应用设置，如温度单位、通知设置等

## 项目结构

```
天气App/
├── index.html (主入口文件，展示所有页面)
├── pages/ (页面文件夹)
│   ├── home.html (首页)
│   ├── details.html (天气详情页)
│   ├── cities.html (城市管理页)
│   ├── search.html (搜索页)
│   └── settings.html (设置页)
├── css/ (样式文件夹)
│   └── style.css (全局样式)
├── js/ (脚本文件夹)
│   ├── components.js (组件脚本)
│   ├── template.js (模板脚本)
│   ├── generator.js (页面生成脚本)
│   └── app.js (应用脚本)
└── img/ (图片文件夹)
    └── sunny-bg.jpg (背景图片)
```

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **FontAwesome** - 图标库
- **JavaScript** - 交互功能

## 使用说明

1. 打开 `index.html` 文件查看所有页面预览
2. 点击各个页面可以查看详细功能和交互
3. 如需修改页面内容，可以直接编辑对应的HTML文件
4. 如需修改样式，可以编辑 `css/style.css` 文件

## 设计特点

- 符合iOS设计规范的界面元素
- 精美的天气图标和直观的数据展示
- 流畅的页面切换和交互效果
- 响应式设计，适配不同尺寸的移动设备
- 高保真UI元素，接近实际App的视觉效果

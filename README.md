# AI UI 收集展示平台

这个项目收集了多种常见应用类型的UI原型设计，包括电商、外卖、音乐、健身、新闻、社区论坛等多种类型的应用界面。每个应用都是基于HTML、CSS和JavaScript构建的高保真原型，可以交互体验。

## 项目概述

- **项目目的**：展示各类应用的UI设计和前端实现，为设计师和开发者提供参考
- **技术栈**：HTML5、CSS3、JavaScript、TailwindCSS
- **原型特点**：高保真、可交互、响应式设计

## 包含的应用类型

此收藏包含以下应用原型：

- 银行App
- 语音房直播App
- 语音直播App
- 社区团购App
- 元宇宙App
- 电商App
- 游戏攻略App
- 情侣App
- 音乐App
- 社区论坛App
- 外卖点餐App
- 新闻App
- 健身App
- more...

## 使用方法

1. 克隆此仓库到本地:
```bash
git clone https://github.com/talentedman/ui-ai-generate.git
```

2. 使用浏览器打开对应应用文件夹中的`index.html`文件，或直接访问导航页`index.html`

3. 每个应用都可以独立运行，展示完整的用户界面和交互体验
